HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element can be given the child element Title which shows up on the tab of the webpage.
  • The body element can be further split into the header, main and footer of the webpage.
  • ul tag can be used to start a unordered list with each bullet point being separated with li tags.
  • Section tag can be used to break up things into sections.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The font-family attribute allows you change font of the text.
  • The color attribute allows you to change color of the text.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: commits all files in the working branch.
  • git commit -m "message": This commits all the changes to our personal repository.
  • git pull origin branch-name: pulls from GitHub repository to your local computer for the chosen branch.
  • git push origin branch-name : updates data to your GitHub repository for the chosen branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Variables can be declared using var, let, const.
  • Declaring a variable with var is not recommended except for supporting old browsers.
  • Declaring a variable with let makes variable flexible and changeable later.
  • Declaring a variable with const makes it a constant that can’t be changed later.
  • You can declare functions and call them up later with functionName().
